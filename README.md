`python_libstorj`
===

Dependencies
---
+ python 2.7
+ [virtualenv](https://virtualenv.pypa.io/en/stable/installation/)


Environment Setup
---
1. Setup a virtualenv
    ```
    virtualenv env
    ```
1. Activate your virtualenv
    ```
    . env/bin/activate
    ```

Build
---
Run the build shell script:
```
./build.sh
```
